# Meeko-Inspired Design Guidelines

## Visual Direction

SourceCred News uses a pastel editorial portfolio aesthetic inspired by the provided Meeko screenshots. It must feel clearly different from:

- ProofBounty: bright white/lime agency SaaS.
- AdProof Escrow: dark emerald grid revenue site.

This project should feel like a playful verification desk: soft lavender backgrounds, black hairline borders, rounded portfolio cards, doodle-style labels, and large calm typography.

## Palette

- Lavender background: `#dedfff`
- Paper white: `#fffefa`
- Ink: `#242326`
- Muted ink: `#56525f`
- Blush card: `#fff0ef`
- Mint card: `#dff9f2`
- Butter card: `#fff6cf`
- Sky card: `#e3f2ff`
- Outline: `#242326`
- Soft shadow: `rgba(36, 35, 38, 0.12)`

## Typography

- Use a modern rounded sans-serif stack.
- Hero: 72-88px desktop, bold, tight line-height, negative letter spacing.
- Section heading: 48-56px.
- Body: 18-22px with comfortable line-height.
- Card labels: 14px uppercase or compact eyebrow text.

## Layout Patterns

- Floating rounded top nav with black border, white fill, and icon buttons.
- Centered oversized hero with a circular mark inserted inline.
- Tilted pill annotations around hero, like editorial stickers.
- Large bordered project-style panels that pair explanation and UI preview.
- Process cards stacked on the right with pastel fills and black outlines.
- Stats in pill-shaped white capsules with outlined display numerals.

## Interaction

- Buttons lift with tiny shadow changes.
- Cards can rotate subtly on hover.
- Contract console should be framed like a design case study, not a dashboard.
- Avoid dark gradients, neon green, or generic dashboard cards.
